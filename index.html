Kvitteringsvakt

Sist oppdatert: 25. mars 2026

Kort oppsummert: Kvitteringsvakt samler ikke inn noen data om deg. All informasjon lagres lokalt på din enhet og i din private iCloud-konto. Vi bruker ingen analyse- eller sporingsverktøy, og deler aldri data med tredjeparter.

Hvilke data behandles

Kvitteringsvakt lar deg skanne og lagre kvitteringer for å holde oversikt over kjøp og garantier. Følgende data kan bli lagret:

  • Bilder av kvitteringer
  • Kjøpsinformasjon (butikk, dato, beløp, produktnavn)
  • Garantiperioder og påminnelser
  • Tekst hentet fra kvitteringer via OCR

All denne informasjonen lagres utelukkende på din enhet ved hjelp av SwiftData. Ingenting sendes til våre servere — vi har ingen.

OCR og tekstgjenkjenning

Når du skanner en kvittering, bruker appen Apples Vision-rammeverk til å lese tekst fra bildet. All tekstgjenkjenning skjer lokalt på enheten din. Ingen bilder eller tekst sendes til eksterne tjenester.

iCloud-synkronisering

Dersom du har iCloud aktivert, kan dataene dine synkroniseres mellom enhetene dine via Apples CloudKit. Denne synkroniseringen håndteres av Apple, og dataene er beskyttet av Apples personvernregler og kryptering.

Vi har ikke tilgang til dataene dine i iCloud. Les mer i Apples personvernerklæring.

Face ID

Kvitteringsvakt tilbyr valgfri applåsing med Face ID eller Touch ID. Biometriske data håndteres utelukkende av iOS og lagres i enhetens Secure Enclave. Appen mottar bare et ja/nei-svar fra systemet — vi har aldri tilgang til ansiktsdata eller fingeravtrykk.

Søk på enheten

Appen bruker CoreSpotlight slik at du kan finne kjøpene dine via iOS-søk. Denne indeksen lagres lokalt på enheten og deles ikke med noen.

Ingen sporing eller analyse

Kvitteringsvakt inneholder:

  • Ingen analyseverktøy (analytics)
  • Ingen sporings-SDK-er
  • Ingen annonsenettverk
  • Ingen tredjepartsbiblioteker som samler data

Vi sporer ikke hvordan du bruker appen, og vi samler ikke inn noen form for brukerdata.

Deling med tredjeparter

Vi deler ingen data med tredjeparter. Punktum.

Sletting av data

Siden all data lagres lokalt på din enhet og i din iCloud-konto, har du full kontroll. Du kan:

  • Slette enkeltkvitteringer direkte i appen
  • Slette all data via appens innstillinger
  • Avinstallere appen for å fjerne all lokal data
  • Administrere iCloud-data via Innstillinger på enheten

GDPR og dine rettigheter

Siden vi ikke samler inn eller lagrer noen personopplysninger på våre servere, er det ingen data hos oss å få innsyn i, rette eller slette. All data er din, og den forblir på din enhet og i din iCloud-konto.

Du har full kontroll over dataene dine til enhver tid, i tråd med EUs personvernforordning (GDPR).

Barns personvern

Kvitteringsvakt samler ikke bevisst inn data fra barn. Siden ingen personopplysninger forlater enheten, behandler vi ikke data fra noen aldersgruppe.

Endringer i personvernerklæringen

Dersom vi oppdaterer denne erklæringen, vil endringene publiseres på denne siden med ny dato. Ved vesentlige endringer vil vi informere via appen.
